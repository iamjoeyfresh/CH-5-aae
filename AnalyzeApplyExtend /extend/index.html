Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

you can use the position style rule to create a "stiky" effect to keep your item/box/content on the page when scrolling.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

you can use relative postion to change the postion on the page relative to what device you are viewing on it will change for different ratios but stay in the same place for all of them.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

the fixed position property is used to keep content in a fixed postion no matter where you are on the page

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

the absolute postion is very simular to the fixed position however it will stay fixed to the closet parent element instead of the page.
